# Nexly — Your Personal Internet

Stop doomscrolling. Nexly shows you only content
that matches your personal life goals.

## What It Does
Type your goals — Learn Python, Build a Startup,
Lose Weight — and Nexly pulls content from Reddit,
GitHub, Hacker News, Medium, Dev.to and News
filtered specifically for your goals.


## Features
- Goal-based content ranking
- 6 content sources in one feed
- Infinite scroll with no duplicates
- Dark and light mode with pixel-fall animation
- No account needed
- No ads or tracking
- Free forever

Expo · React Native · Reddit API · GitHub API · NewsAPI
